# AgentForge Flywheel

Monitor the compounding flywheel — the system that makes agents get smarter, faster, and more autonomous over time.

## Subcommands

- `health` — Dashboard showing all 4 flywheel components and their active/inactive status
- `insights` — Show actionable insights generated by the meta-learning engine
- `autonomy` — Show agent autonomy tiers and promotion/demotion history
- `skills` — Show capability inheritance: which skills have been propagated between agents
- `velocity` — Show sprint velocity trend and acceleration ratio

## What to Do

1. Import `MetaLearningEngine` from `src/flywheel/meta-learning-engine.ts`
2. Import `AutonomyGovernor` from `src/flywheel/autonomy-governor.ts`
3. Import `CapabilityInheritance` from `src/flywheel/capability-inheritance.ts`
4. Import `FlywheelMonitor` from `src/flywheel/flywheel-monitor.ts`

### health
Call `monitor.getFlywheelHealth()` and display 4 components:
- **Meta-learning** — active when ≥2 insights generated
- **Graduated autonomy** — active when ≥1 agent promoted
- **Capability inheritance** — active when ≥1 skill propagated
- **Velocity acceleration** — active when sprint velocity ratio > 1.0

### insights
Call `engine.generateInsights()` — shows patterns with success rates and recommendations:
- `recommend` — high-success patterns to adopt
- `avoid` — low-success patterns to stop using
- `investigate` — patterns needing more data

### autonomy
Call `gov.getPromotionHistory()` and list agents by tier:
- Tier 1: Supervised (15 min timeout)
- Tier 2: Assisted (30 min timeout)
- Tier 3: Autonomous (60 min timeout)
- Tier 4: Strategic (120 min timeout)

Promotion requires 5 consecutive successes. Demotion triggers after 3 consecutive failures.

### skills
Call `ci.getPropagationHistory()` — shows which skills have been inherited, source → target agent, proficiency scaling (60% of source).

### velocity
Call `monitor.getVelocities()` and `monitor.getVelocityRatio()` — shows tasks completed per sprint and acceleration trend. Target: ratio > 1.05 for healthy compounding.
