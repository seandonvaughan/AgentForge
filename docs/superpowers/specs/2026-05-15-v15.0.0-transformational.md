# v15.0.0 — Transformational + Foundational

Sprint goal: turn AgentForge from a system with one working hot-path into a robust autonomous platform that engages the entire 44-agent team across foundational architecture, data subsystems, dashboard surfaces, security, and ops. This sprint deliberately spans every category in `team.yaml` (strategic, implementation, quality, utility).

**Calibration context for the scorer** (real medians from last 7 cycles):
- ci / typecheck / release-gate: ~$0.75
- fix (specific file + function target): ~$1.10
- chore (verify / document / move): ~$0.55
- feature (wire up dashboard route): ~$1.65
- migration / refactor: ~$2.00
- test (add unit / integration coverage): ~$0.90
- doc (README / CHANGELOG / spec): ~$0.55
- e2e (Playwright): ~$2.50
- security (audit + fix): ~$1.00

DO NOT estimate items at $5+ unless they span multiple subsystems with new architecture.
